# Changelog

## v1.0.0 — 2026-04-12

Initial public release.

- Hyprland config with 4 named persistent workspaces (ALPHA, MIKE, DELTA, ECHO)
- Per-workspace wallpaper switching via `wallpaper.sh`
- Waybar status bar with dark steel blue theme
- Kitty terminal with custom color scheme
- Fish shell + Starship prompt
- Fastfetch system report with ops center styling
- btop with custom `sigint-01` theme
- Neovim colorscheme plugin config
- Hyprlock lock screen
- greetd + tuigreet display manager
- Flameshot screenshot tool config
- Custom wallpaper generator (`generate_wallpapers.py`) — 4 PNG wallpapers built with Python/Pillow
- SVG wallpaper source files for ALPHA, MIKE, DELTA, ECHO workspaces
- `install.sh` bootstrap script — packages, symlinks, wallpapers, services, shell
- Hardened `sshd_config` (optional, opt-in)
- GitHub Pages site with boot sequence, typing code background, screenshot gallery
